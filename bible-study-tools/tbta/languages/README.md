# TBTA Language Resources

**Purpose**: Document how TBTA features apply to specific language families and provide guidance for Bible translators working in 1,000+ languages.

---

## Language Family Highlights

### Austronesian Languages (172 languages)
**Critical Features**: Trial number, Clusivity
- Example: Kilivila (Papua New Guinea) - requires trial for Genesis 1:26 Trinity reference

### Philippine Languages (468 languages)
**Critical Features**: Participant Tracking, Discourse markers
- Example: Ilokano - distinguishes kami (exclusive) vs tayo (inclusive) for Acts 15:25

### East Asian Languages (Japanese, Korean, Chinese)
**Critical Features**: Time Granularity, Speaker Demographics, Proximity System
- Example: Japanese - requires これ/それ/あれ demonstrative distinctions and honorific levels

### Native American Languages (468 languages)
**Critical Features**: Participant Tracking, Number systems
- Example: Ambiguous referents in Genesis 4:8 affect pronoun selection
