## Critical Patterns Discovered

### 1. Value Disambiguation Patterns

#### a. 'less' (L) vs Comparative (C)

**'less' (L)**:
- Progressive decrease, no explicit comparison
- GEN 8:1-5: "Waters became less" (flood receding)
- Pattern: Intransitive decrease

**Comparative (C)**:
- Explicit comparison: "X is less than Y"
- Pattern: Transitive comparison

#### b. 'too' (T) vs Comparative (C)

**'too' (T)**:
- Beyond capacity or ability
- GEN 4:13: "Too great to bear"
- RUT 1:11: "Too old to bear sons"
- Pattern: Excessive degree → inability

**Comparative (C)**:
- Relative degree
- GEN 19:31: "Older than custom"
- Pattern: Simple comparison

#### c. Superlative of 2 (s) vs Comparative (C)

**Superlative of 2 (s)**:
- Selection from exactly 2 items
- GEN 27:15: "The older (of 2 sons)"
- GEN 29:18: "The younger (of 2 daughters)"
- Pattern: Definite article + selection from pair

**Comparative (C)**:
- General comparison between 2+ items
- Pattern: No selection implied

### 2. Mixed Annotation Patterns

Same constituent can have multiple degree markings:
- GEN 18:11: "old" = Intensified + 'too'
- EXO 10:14: "great" = Superlative + Comparative
- EXO 11:6: "loudly" = Intensified + Comparative (2x)

**Pattern**: Each semantic aspect annotated separately!

### 3. Hebrew vs Greek Patterns

#### Hebrew מִן Constructions:

Can map to multiple degrees based on context:
- **Comparative**: GEN 7:20 (rising waters, "15 cubits above")
- **'too' (T)**: GEN 4:13 ("too great to bear")
- **'less' (L)**: GEN 8:1 (receding waters, "becoming less deep")

**Trigger**: Context + verb semantics determines degree!

#### Greek Intensifiers:

- πολλῷ, πολλῶν → Intensified Comparative (i)
- ὑπερβολή → Extremely Intensified (E)
- Synthetic comparatives (μείζων) → Comparative or Superlative (context-dependent)

### 4. Discourse-Level Patterns

#### Poetic Contexts (Psalms, Exodus 15):
- Repetition with same degree
- Parallelism (multiple constituents)
- Clustering of Intensified/Extremely Intensified

#### Narrative Contexts (Genesis):
- Progressive descriptions: GEN 8:1-5 ('less' sequence)
- Ranking contexts: GEN 43:33 (oldest to youngest)

#### Epistolary Contexts (Ephesians):
- Theological clustering: EPH 1:6-18 (Extremely Intensified × 5)
- Emphasis on abundance and grace

### 5. Encoding Variations

**Literal values use varying quote styles**:
- '''least''' (triple quotes): MAT 5:19
- 'least' (single quotes): JDG 6:15, 1SAM 9:21, EST 1:5
- 'less' (single quotes): GEN 8:1, 8:3, 8:5
- 'too' (single quotes): GEN 4:13, RUT 1:11, etc.

**Pattern**: Parser must handle both triple and single quotes!

## Key Insights for Algorithm Development

### 1. Value Inventory is COMPLETE
- All TBTA degree values now documented
- No more unknown values (q confirmed non-existent)
- Can build comprehensive value list for v2.1+

### 2. Context is CRITICAL
- Same morphology → different degrees (Hebrew מִן)
- Same word → multiple degrees (mixed annotations)
- Context clues: verb semantics, discourse type, set size

### 3. Source Language Matters
- Hebrew: Complex מִן patterns, definiteness matters
- Greek: Synthetic forms, prefix intensifiers, theological clusters
- Can't treat both identically

### 4. Literal Values are REAL
- Not encoding errors - legitimate TBTA values
- Quote style varies (triple vs single)
- Must parse flexibly

### 5. Edge Cases are COMMON
- Mixed annotations: 20+ instances in 100 verses
- Complex verses (3+ constituents): 10+ instances
- Not exceptional - normal variation!
