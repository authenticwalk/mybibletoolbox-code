# Other Language Families - Overview

This directory provides comprehensive research on all language families in the myBibleToolbox dataset that are NOT Austronesian, Trans-New Guinea, Niger-Congo, or Indo-European.

**Total languages covered**: 468 languages across 70+ families

## Purpose

Understanding grammatical features of diverse language families is critical for:
- Bible translation accuracy
- AI-assisted translation technology
- Linguistic typology awareness
- Cross-linguistic feature mapping

## Key Typological Features

### Morphological Types
- **Polysynthetic**: Uto-Aztecan, Eskimo-Aleut, Eyak-Athabaskan, Totonacan, Mixe-Zoquean
- **Agglutinative**: Quechuan, Dravidian, Turkic, Tupian
- **Isolating to Polysynthetic range**: Sino-Tibetan (most structurally diverse family)
- **Simplified morphology**: Creole languages

### Alignment Systems
- **Ergative-Absolutive**: Mayan, Australian (many), Eskimo-Aleut, Panoan
- **Split Ergativity**: Australian (by person/animacy)
- **Inverse Systems**: Mixe-Zoquean (by animacy/topicality)

### Tone Systems
- **All tonal**: Otomanguean (oldest tonal family)
- **Mostly tonal**: Afro-Asiatic (except Berber, Semitic), Sino-Tibetan (variable), Nilo-Saharan (2-4 tones)

### Special Grammatical Categories
- **Evidentiality**: Quechuan (3-term), Tucanoan (5-term in Tuyuca), Cariban, Panoan
- **Noun Classification**: Arawakan (classifiers), Sepik (10 classes), Torricelli (10+ classes)
- **Gender Systems**: Arawakan (masculine/feminine), Afro-Asiatic (masculine/feminine)

### Word Order Patterns
- **VSO**: Otomanguean, Mayan (VOS/VSO)
- **SOV**: Quechuan, Tupian, Turkic, Dravidian, Sino-Tibetan (mostly)
- **SVO**: Torricelli (unusual for Papuan), Creole languages
- **Free word order**: Australian, Dravidian (relatively free)
- **OVS**: Cariban (Hixkaryana - rare globally)

### Unique Features
- **Root-and-Pattern Morphology**: Afro-Asiatic (consonantal roots)
- **Vowel Harmony**: Turkic (nearly universal)
- **Kinship Grammar**: Australian (dual pronouns vary by relationship)
- **Body Part Incorporation**: Totonacan (delimits locus of affect)
- **Avoidance Registers**: Australian (special vocabulary for certain relatives)

## Translation-Relevant Insights

### High-Complexity Features
1. **Evidentiality**: Obligatory marking of information source (Quechuan, Tucanoan, Panoan)
2. **Tonal accuracy**: Critical for meaning in all Otomanguean, most Afro-Asiatic branches
3. **Polysynthesis**: Single words express entire propositions
4. **Ergativity**: Different subject/object conceptualization than accusative languages
5. **Noun classes**: Semantic categorization affects agreement throughout sentences

### Low-Resource Challenges
- Otomanguean: Models drastically underperform due to complex tone + inflection
- Many Papuan families: Limited English-language documentation
- Amazonian families: Under-documented, require specialized sources
- Language isolates: No comparative method available

## Research Status

### Well-Documented (substantial online sources)
Otomanguean, Mayan, Afro-Asiatic, Australian, Uto-Aztecan, Sino-Tibetan, Quechuan, Tucanoan, Dravidian, Turkic, Eskimo-Aleut

### Under-Documented (require specialized sources)
- Most Papuan families (Border, Jean, Ramu-Lower Sepik, etc.)
- Small Amazonian families (Witotoan, Jivaroan, Guajiboan, Barbacoan, Tacanan)
- Many language isolates
- Single-language families

### Recommended Research Priorities
1. Papuan languages (SIL publications, dissertations)
2. Amazonian languages (Brazilian archives, missionary sources)
3. Individual language isolates (field linguistics sources)
4. Translation technology implications (MT, evidentiality, polysynthesis challenges)
