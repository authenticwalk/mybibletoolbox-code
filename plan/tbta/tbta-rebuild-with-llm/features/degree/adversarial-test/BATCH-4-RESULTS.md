## Key Patterns from Batch 4

### Pattern 1: s (Superlative of 2) vs C (Comparative)

**Triggers for s**:
- Exactly 2 items in context (2 sons, 2 daughters)
- Selection/ranking implied ("the older one", "the younger one")
- Hebrew: Definite article + age term patterns

**Triggers for C**:
- General comparison (no specific set size)
- Relative description ("older than", "younger than")

### Pattern 2: Repetition Handling

TBTA annotates **each instance separately**:
- EXO 11:6: "loudly" √ó 3, all annotated
- EXO 1:9: "much-many" √ó 2, both annotated
- Each instance can have different degrees!

### Pattern 3: NT Greek Patterns

Greek uses similar intensification strategies:
- 'too' appears in Greek (ACT 2:8)
- Extremely Intensified clusters in Ephesians 1
- Standard C, S, I patterns parallel Hebrew

### Pattern 4: Poetic Intensification

Poetic contexts (Psalms, Exodus 15) use:
- Repetition with same degree
- Parallelism (multiple constituents in parallel)
- Intensified and Extremely Intensified for emphasis

### Pattern 5: Context-Dependent Annotation

Same word, different degrees based on context:
- EXO 10:14: "great" = Superlative + Comparative
- GEN 18:11: "old" = Intensified + 'too'
- Context determines which aspect is marked
