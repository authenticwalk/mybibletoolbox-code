## Key Learnings

### What Worked in Cycle 2

1. **Grammatical Pathway Specialization:**
   - Tailoring extraction to word type (grammatical vs theological) yields richer results
   - Pronouns benefit from morphology/syntax focus, not semantic/theological focus

2. **Comprehensive Morphology Extraction:**
   - All 49 forms documented (vs top 5) provides complete paradigm
   - Minimal additional time investment (+10 min) for major richness gain

3. **Syntax/Collocation Documentation:**
   - Real usage patterns more valuable than theoretical definitions
   - Verse examples ground abstract grammatical concepts

4. **Pedagogical Integration:**
   - Learning challenges + resolutions add practical teaching value
   - Mounce's student focus complements scholarly lexicons

5. **Diachronic Frequency Focus:**
   - Frequency shifts (not semantic development) appropriate for grammatical words
   - 4-stage trajectory shows clear evolution from Epic to Koine

### What Could Be Improved

1. **Extended Form Extraction:**
   - Could document all 49 forms individually (currently top 15)
   - Diminishing returns for ultra-low frequency forms
   - Would add ~15 minutes

2. **Comparative Pronoun Analysis:**
   - Could contrast with οὗτος (G3778), ἐκεῖνος (G1565)
   - Would show grammatical parallels and distinctions
   - Would add ~20 minutes

3. **Discourse Analysis:**
   - Could analyze pronoun usage in extended discourse (e.g., John 1:1-18)
   - Would show anaphoric reference patterns
   - Would add ~30 minutes

**Decision:** Current depth optimal for project scope. Further expansion optional.
