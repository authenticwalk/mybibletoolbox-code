## Lessons Learned

### Grammatical Pathway Advantages

1. **Specialization Works:**
   - Grammatical words benefit from morphology/syntax focus (not semantic/theological)
   - Tailoring extraction to word type yields richer results

2. **Comprehensive Morphology High ROI:**
   - 227% increase in forms (+10 min investment)
   - Complete paradigm valuable for students/translators

3. **Syntax > Semantics for Function Words:**
   - Real usage patterns (syntax, collocations) more valuable than abstract definitions
   - Grammatical function words lack theological depth (HELPS/TDNT usually absent)

4. **Pedagogical Value Critical:**
   - Learning challenges address practical teaching needs
   - Student-focused insights complement scholarly lexicons

5. **Diachronic Frequency Appropriate:**
   - Frequency shifts (not semantic development) suitable for pronouns
   - Shows grammaticalization processes (Epic emphatic → Koine standard pronoun)

### Validation Insights

1. **Pathway-Specific Criteria:**
   - Grammatical pathway criteria (morphology depth, syntax patterns) better fit than generic criteria (TDNT, BDAG)
   - Level 3 redefined for grammatical words (morphology/syntax vs cross-refs)

2. **100% Achievable:**
   - All sources available in ATTRIBUTION.md
   - Inline citations straightforward
   - No fabrication maintained
