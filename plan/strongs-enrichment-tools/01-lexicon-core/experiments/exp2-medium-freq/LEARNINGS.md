### 2. Controversy Documentation Success ✓

**Target:** Document and refute "dunamis = dynamite" false etymology

**Sources Found:**
1. **scribalcafe.wordpress.com** - Pastor Mark Drinnenberg's refutation
2. **wordstudytools.com** - Linguistic precision refutation

**Key Refutation Points Documented:**
- **Time-frame fallacy:** 1,800 year gap between NT (1st c.) and Nobel's invention (1867)
- **Semantic incompatibility:** Explosions sudden/momentary vs δύναμις enduring/sustained
- **Modern Greek evidence:** Uses different words (ἀνατινάζω, έκρηξη) for explosions
- **Scientific naming:** Nobel followed convention of Greek/Latin terms, doesn't define ancient meaning
- **Scholarly terminology:** "Semantic back-formation and reconstruction"

**Learning:** Controversy documentation is highly feasible with WebSearch + WebFetch. False etymologies leave clear refutation trails online. This validates including controversy detection as a regular lexicon-core task.

## Methodology Refinements

### What Worked Well

1. **Parallel web fetching:** Fetching BibleHub, StudyLight, BLB simultaneously saved time
2. **Controversy-first search:** Using WebSearch for controversy before WebFetch yielded exact refutation sources
3. **Schema adherence:** Following Exp 1 schema exactly ensured consistency
4. **Base file reference:** Noting what's already in base file prevented duplication
5. **Citation discipline:** Inline {source} tags throughout maintained traceability

### What to Improve

1. **Controversy search timing:** Could search for controversy BEFORE main web fetches to inform extraction strategy
2. **TDNT content extraction:** BLB provides reference number but not content - may need TDNT-specific fetch for full treatment
3. **Trench content extraction:** Similar to TDNT - reference found but full text requires separate source
4. **Papyri systematization:** Papyri evidence found ad-hoc in StudyLight - could benefit from dedicated papyri source
5. **LXX usage depth:** LXX occurrence count (383) noted but semantic analysis minimal - could explore LXX usage patterns more

### Process Optimizations for Future Experiments

1. **Pre-flight controversy check:**
   - Before main extraction, run WebSearch for "{lemma} controversy etymology false"
   - If hits found, prioritize controversy documentation in extraction strategy

2. **Scholarly reference extraction:**
   - When TDNT/Trench references found, note for potential follow-up extraction
   - Consider whether reference NUMBER sufficient or full CONTENT needed
   - Balance completeness vs time investment

3. **Source prioritization for word types:**
   - **Theological terms:** HELPS + Thayer + TDNT + Trench (high value)
   - **Grammatical terms:** Morphology + syntax + usage statistics (HELPS likely absent)
   - **Rare words:** Etymology + classical usage + hapax analysis

4. **Validation checkpoints:**
   - Run Level 1 validation DURING extraction (not just after)
   - Ensures inline citations added in real-time
   - Catches missing sources before file completion

## Key Learnings for Tool Development

### 1. Word Frequency Sweet Spot

**Finding:** Medium-frequency theological terms (50-500 occurrences) provide RICHEST unique data extraction.

**Implications:**
- Prioritize medium-frequency words in lexicon-core tool development
- High-frequency functional words (pronouns, articles, particles) yield less unique web data
- Low-frequency rare words (<50) may lack web source coverage

**Recommendation:** Stratify Strong's numbers by frequency:
- **Tier 1 (50-500 occurrences):** Full extraction - highest ROI
- **Tier 2 (10-49 occurrences):** Targeted extraction - focus on etymology/classical
- **Tier 3 (500+ occurrences):** Basic extraction - likely functional words
- **Tier 4 (2-9 occurrences):** Minimal extraction - web sources may be sparse

### 2. Theological vs Grammatical Terms

**Finding:** Theological/lexical terms receive substantially richer web source treatment than grammatical terms.

**Theological Term Advantages:**
- HELPS Word-studies present
- TDNT treatment available
- Trench's Synonyms coverage
- Deeper semantic analysis
- Richer synonym networks
- Scholarly cross-references abundant

**Grammatical Term Characteristics:**
- HELPS often absent
- TDNT/Trench unavailable
- Focus on morphology/syntax
- Limited semantic range
- Fewer scholarly discussions

**Recommendation:** Develop parallel extraction strategies:
- **Theological pathway:** Full semantic + scholarly + controversy
- **Grammatical pathway:** Morphology + syntax + usage stats

### 3. Controversy Detection Value

**Finding:** False etymology detection and refutation adds significant scholarly value.

**G1411 Case Study:**
- Popular sermon cliché identified
- Clear refutation sources found
- Scholarly terminology documented
- Pedagogical value high (teaches etymological fallacy awareness)

**Recommendation:**
- Make controversy detection standard lexicon-core feature
- Search pattern: "{lemma} false etymology controversy refutation"
- Common false etymology types:
  - English back-formation (like dynamite)
  - Folk etymology (popular but wrong)
  - Denominational distinctives (theological eisegesis)

### 4. Synonym Network Importance

**Finding:** English translation collapse obscures Greek semantic distinctions.

**G1411 Example:** English "power" translates 5+ distinct Greek concepts:
- δύναμις (inherent ability)
- ἰσχύς (physical strength)
- κράτος (manifested power)
- ἐξουσία (authority)
- ἐνέργεια (operative power)

**Theological Impact:** Conflating these terms creates doctrinal confusion (authority vs ability, inherent vs delegated, capacity vs manifestation).

**Recommendation:**
- Make synonym network extraction standard lexicon-core task
- Document semantic distinctions with theological precision
- Highlight English translation inadequacies

### 5. Diachronic Dimension

**Finding:** Classical → Koine development shows semantic specialization for theological terms.

**G1411 Trajectory:**
- **Homer:** Bodily strength (broad physical)
- **Classical:** Military, philosophical, mathematical (diversification)
- **Papyri:** Economic + magical/religious (transitional)
- **NT/Koine:** Theological specialization (focused miraculous/divine)

**Pattern:** Semantic narrowing toward religious/theological application in Koine period.

**Recommendation:**
- Document Classical breadth (LSJ)
- Trace papyri transition (documentary evidence)
- Identify NT specialization (theological focus)
- This creates "semantic funnel" visualization
