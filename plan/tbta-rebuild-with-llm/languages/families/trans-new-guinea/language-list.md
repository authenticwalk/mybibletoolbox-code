## Geographic Distribution

- **Primary location**: Papua New Guinea (vast majority)
- **Secondary locations**: Indonesia (West Papua/Papua provinces), particularly for languages like Yale (kkl), Nggem (nbq), and Blagar (beu)
- **Region**: Primarily the interior highlands and mountains of New Guinea island

## Notes

This list represents languages with available data in the myBibleToolbox dataset. The total Trans-New Guinea family is estimated to include 300-500 languages total.

For information about which sub-family each language belongs to, see [sub-families.md](sub-families.md).
