# Historical and Comparative Context

This document provides historical context for the Austronesian language family, including Proto-Austronesian reconstruction, migration patterns, and language contact effects.

## Proto-Austronesian Reconstruction

### The Austronesian Comparative Dictionary (ACD)

**Creators**: Robert Blust and Steve Trussel

**Scope**:
- **119,768 individual words** from 1,022 language varieties
- **8,161 cognate sets** with reconstructions
- **Nine hierarchically ordered protolanguages**: Reconstructions at multiple levels

**Current Status**: Maintained by Alexander D. Smith (after Blust's passing in 2022)

**Online Access**:
- https://acd.clld.org/
- https://www.trussel2.com/ACD/

### Proto-Austronesian Features (Reconstructed)

**Four-voice system**:
- *-um-* (Actor Focus)
- *-en* (Patient Focus)
- *-an* (Locative Focus)
- *Si-* (Instrumental Focus)

**Accusative alignment**: Proto-Austronesian was accusative (preserved in Rukai)

**Reanalysis**: "Non-accusative alignment found in other Austronesian languages resulted first from the reanalysis of irrealis clause types"

### Significance for Bible Translation

- Comparative data helps identify inherited features vs. innovations
- Can inform decisions about which structures are most "natural" in related languages
- Etymological information useful for key term studies across languages

**Source**: [Nature: Austronesian Comparative Dictionary; Wikipedia: Proto-Austronesian]
