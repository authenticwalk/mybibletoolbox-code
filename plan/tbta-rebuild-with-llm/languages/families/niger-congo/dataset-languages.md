# Niger-Congo Languages in Our Dataset

## Summary Statistics

- **Total Languages**: 94 Niger-Congo languages
- **Geographic Distribution**: West, Central, East, and Southern Africa
- **Countries Represented**: 22 countries
- **Sub-families Present**: Bantu, Atlantic, Gur, Kwa, Mande, Adamawa-Ubangi, Kru, Ijoid, Benue-Congo

#### Wolof
