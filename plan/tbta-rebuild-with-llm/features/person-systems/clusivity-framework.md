# Clusivity Framework for Bible Translation

## Executive Summary

This framework provides a systematic approach for determining inclusive vs exclusive readings of first-person plural pronouns (we/us/our) in Bible translation for languages that grammatically distinguish clusivity.

**Key Finding**: Theological and discourse factors are more determinative than grammatical analysis alone. A hierarchical decision process based on ontological distinctions, capability analysis, and discourse function yields highly accurate predictions.

## The Five-Level Analysis Framework

### Level 1: Ontological Analysis (Highest Priority)
**Question**: What is the fundamental nature of speaker and addressee?

| Speaker | Addressee | Result |
|---------|-----------|---------|
| Divine | Human | **EXCLUSIVE** |
| Divine | Divine | Context-dependent |
| Human | Divine (prayer) | **EXCLUSIVE** (God not in "we") |
| Human | Human | Continue to Level 2 |

### Level 2: Capability Analysis
**Question**: Can the addressee participate in the stated action?

- **Cannot participate** → EXCLUSIVE
  - Eyewitness testimony to non-witnesses
  - Specialized roles (apostolic authority)
  - Past events addressee wasn't part of

- **Can participate** → Continue to Level 3

### Level 3: Group Identity Analysis
**Question**: Do speaker and addressee belong to the same group?

- **Different groups** → EXCLUSIVE
  - Ethnic distinctions ("We Jews" to Gentiles)
  - Religious distinctions ("We believers" to unbelievers)
  - Geographic distinctions ("We from Jerusalem")

- **Same group** → Continue to Level 4

### Level 4: Discourse Function Analysis
**Question**: What is the rhetorical purpose?

| Function | Typical Result |
|----------|---------------|
| Establishing authority | EXCLUSIVE |
| Building solidarity | INCLUSIVE |
| Teaching/exhorting | Context-dependent |
| Testimony | EXCLUSIVE |
| Invitation | INCLUSIVE |
| Confession | INCLUSIVE |

### Level 5: Grammatical Cues
**Question**: Are there explicit grammatical markers?

- Vocative creating separation → EXCLUSIVE
- "All of us" with shared trait → INCLUSIVE
- Demonstrative distinctions → EXCLUSIVE

## Genesis Divine Plural Pattern

All instances of divine plural speech in Genesis follow consistent patterns:

| Verse | Text | Interpretation | Clusivity |
|-------|------|----------------|-----------|
| Gen 1:26 | "Let us make man" | Divine council/Trinity | **EXCLUSIVE** |
| Gen 3:22 | "Like one of us" | Divine assessment | **EXCLUSIVE** |
| Gen 11:7 | "Let us go down" | Divine judgment | **EXCLUSIVE** |

**Pattern**: Divine creative/judicial acts always exclude human participation

## Theological Principles

### Divine Speech Rules
1. Creation acts → EXCLUSIVE (humans can't create themselves)
2. Judgment acts → EXCLUSIVE (humans don't judge themselves)
3. Divine knowledge → EXCLUSIVE (unique divine perspective)

### Prayer Context Rules
1. Addressing God → EXCLUSIVE (God not included in "our")
2. Teaching prayer → May be INCLUSIVE (all believers)
3. Intercessory prayer → Context determines inclusion

### Apostolic Speech Rules
1. Eyewitness testimony → EXCLUSIVE
2. Apostolic authority → Initially EXCLUSIVE
3. General teaching → May shift to INCLUSIVE

## Quick Decision Guide

```
START: Is the speaker divine?
  ├─ YES: Is the action creation/judgment?
  │    ├─ YES → EXCLUSIVE
  │    └─ NO: Is addressee divine?
  │         ├─ YES → Possibly INCLUSIVE (Trinity)
  │         └─ NO → EXCLUSIVE
  │
  └─ NO: Is this prayer to God?
       ├─ YES → EXCLUSIVE (God not in "we")
       └─ NO: Can addressee do the action?
            ├─ NO → EXCLUSIVE
            └─ YES: Same group identity?
                 ├─ NO → EXCLUSIVE
                 └─ YES: What's the purpose?
                      ├─ Authority → EXCLUSIVE
                      ├─ Solidarity → INCLUSIVE
                      └─ Teaching → CHECK CONTEXT
```

## Common Patterns Reference

### Reliably EXCLUSIVE
- Genesis divine plurals (1:26, 3:22, 11:7)
- Prayer addressing God
- Apostolic eyewitness claims
- Ethnic/religious distinctions
- Restricted capability actions

### Reliably INCLUSIVE
- Community worship calls
- Shared faith experiences
- Unity statements
- Universal believers' prayers
- Participation invitations

### Context-Dependent
- Pauline "we" (often layered)
- Teaching passages
- Narrative perspectives
- Prophetic utterances

## Implementation Guidelines

### For Translators
1. **Document decisions** - Keep record of clusivity choices
2. **Maintain consistency** - Same speaker/context = same clusivity
3. **Consider theology** - Decisions have doctrinal implications
4. **Provide notes** - Footnote ambiguous cases

### For AI Systems
1. **Apply hierarchically** - Ontological > Capability > Identity > Function
2. **Flag ambiguities** - Some cases legitimately have dual readings
3. **Track patterns** - Learn from consistent translator decisions
4. **Preserve alternatives** - Document valid alternative readings

## Error Recovery Protocol

When predictions seem incorrect:

1. **Recheck speaker identity** - Who exactly is speaking?
2. **Recheck addressee** - Who is actually being addressed?
3. **Expand context** - Read surrounding verses
4. **Consider layers** - Some passages intentionally ambiguous
5. **Check parallels** - How are similar passages handled?
